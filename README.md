# Vue-webApp Sample
vue webapp 프로젝트 진행시 초반 세팅용으로 작업.

### Install packages
```bash
$ npm install
```

### Start development server
```bash
$ npm run dev
```

### Build for production
```bash
$ npm run build
```

### 주요 패키지 버전
```bash
"vue": "^2.5.17"
"vue-router": "^3.0.1"
"vue-analytics": "^5.16.0" // 구글애널리틱스
"vuetify": "^1.5.21"  // VueUI 프레임워크
"axios": "^0.18.1" // 노드 기반 Http 클라이언트
```
